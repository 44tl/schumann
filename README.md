# Schumann Resonance

Live dashboard tracking the fundamental electromagnetic resonance of the Earth–ionosphere cavity (~7.83 Hz), with real-time spectrograms and solar activity from NOAA / NASA sources.

**Made by 44tl (aka emy)**

Data Collection: [Tomsk Observatory](http://sosrff.tsu.ru/) · [NOAA SWPC](https://www.swpc.noaa.gov/) · [NASA SDO](https://sdo.gsfc.nasa.gov/) · [SOHO/LASCO](https://soho.nascom.nasa.gov/)
